# SW5E Character Sheet

A character sheet web app for Star Wars 5e (SW5E), the D&D 5e conversion ruleset.

- React + TypeScript + Vite
- All data stays local to your device (browser localStorage), with JSON export/import for backups or sharing
- Tracks ability scores, skills, saving throws, combat stats, Force/Tech powers and points, equipment/credits, and freeform notes

## Development

```bash
npm install
npm run dev
```
